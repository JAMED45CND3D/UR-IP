✦ uriplib.js v1.0
Library Core UR-IP
JavaScript library yang menangani semua logika encoding, decoding, drawing, dan error correction untuk simbol SYKLUS. Bisa dipakai di browser maupun Node.js.
API · Encode
URIP.encode(text, mode)
Mengubah teks menjadi bitstream terstruktur dengan header, CRC16, dan Reed-Solomon parity.
// Encode teks const result = SYKLUS.encode("halo dunia", "STANDARD"); result.bits // Array of 0/1 result.mode // "STANDARD" result.crc // CRC16 checksum (number) result.hash // FNV hash → rotasi unik result.totalBits // 450
API · Decode
URIP.decode(bits, mode)
Mengubah bitstream kembali menjadi teks dengan verifikasi CRC dan Reed-Solomon correction.
// Decode bits const result = SYKLUS.decode(bits, "STANDARD"); result.ok // true jika valid result.text // teks yang didecode result.crcOk // CRC verified result.hasError // RS detected error
API · Draw
URIP.draw(canvas, encodeResult, theme, animOffset)
Menggambar simbol UR-IP ke HTML Canvas. Dua arm spiral, pusat Pancer, cardinal dots.
// Draw ke canvas const enc = SYKLUS.encode("test", "LITE"); SYKLUS.draw(canvas, enc, "gold", 0); // Animasi let angle = 0; function loop() { SYKLUS.draw(canvas, enc, "gold", angle); angle += 0.008; requestAnimationFrame(loop); }
Struktur Data
Format Bitstream
Header (32 bit): version(8) · mode index(8) · payload length(16)
Payload: UTF-8 bytes dari teks
CRC16-CCITT (16 bit): verifikasi integritas payload
Reed-Solomon (64 bit): 8 parity symbols untuk koreksi error
Padding: mengisi sisa kapasitas mode

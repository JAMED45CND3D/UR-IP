Library Core UR-IP
JavaScript library yang menangani semua logika encoding, decoding, drawing, dan error correction untuk simbol Triskelion. 3 spiral arm · 0° · 120° · 240° · Nodes sebagai titik bulat.
URIP.encode(text, mode)
const result = URIP.encode("halo", "STANDARD");
result.bits // array bit per arm
result.totalBits // total bits
result.mode // mode yang dipakai
URIP.decode(bits, mode)
const result = URIP.decode(bits, "STANDARD");
result.text // teks yang di-decode
result.ok // CRC valid?
result.crcOk // checksum status
Struktur Simbol Triskelion
3 spiral arm · offset 0°, 120°, 240°
Bit 1 = node besar (radius 6px)
Bit 0 = tidak digambar (background gelap)
Spiral logaritmik: r = a·e^(b·t)
Timing ring + boundary ring sebagai reference
Center Pancer: anchor deteksi
CRC-16 + Reed-Solomon error correction
Filosofi
UR-IP tidak dirancang. UR-IP ditemukan — dari sketsa tangan, dari pertanyaan yang tidak tahu jawabannya, dari simbol yang sudah ada miliaran tahun.

Pusat tidak ikut berputar. Pusat adalah anchor. Yang berputar adalah yang mengorbit.

Tiga arm bukan kebetulan — triskelion adalah geometri minimum untuk redundansi dan error recovery.
